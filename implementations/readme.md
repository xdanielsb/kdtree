## C++
$ g++ -std=c++14 -o bin nameFile.cpp  && ./bin

## Python
$ python nameFile.py

## Java
```sh
$ export JAVA_HOME=/usr/local/jdk1.8.0_111
$ export PATH=$JAVA_HOME/bin:$PATH

$ javac nameFile.java
$ java nameFile
```

## Go
```sh
$ export PATH=$PATH:/opt/go/bin
$ export GOPATH=$HOME/Documents/golang
$ export GOROOT=/opt/go
$ go run nameFile.go
```
