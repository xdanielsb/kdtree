# Simulation

Here is the way to execute the Simulation


## One class

```sh
  $ ./run.sh  N
```
- N is the number of random data that you want

#### One class, N = 10, C = 1
<img src="img-readme/one_class_1.png">

## Multi class

```sh
  $ ./run.sh  N C
```
- N is the number of random data that you want
- C is the number of classes that you want
